What is the Wall?

    Thank you for the download of `the Wall'. The Wall changes background images and/or color at user-set intervals.

    This program is licensed to you under the provisions of the License Agreement included in this file. Use of this program constitutes your acceptance of such Agreement.

    You can configure each image file in such as *.BMP/*.GIF/*.JPG/*.TIF/*.TGA/*.PCX/*.DIB in the Wall. Choose TILED, SIZED or NORMAL during configuration. The Wall writes an extended attribute for each and displays the .BMP as configured.

    This OS/2 application requires VROBJ.DLL, the runtime module from Watcom VX-REXX.
    You can get a copy from...

    • Download VROBJ.DLL 2.1d (vrobj21d.zip)

Installation
    1. Create a subdirectory
        ex. [x:\]md Wall
    2. Move or copy Wall???r.zip to the directory created in action a.
    3. Unzip Wall???r.zip using your favorite zip utility
    4. Run install.cmd from a OS/2 command line.
        ex. [x:\Wall\]install
    5. Now you can return to the WorkPlaceShell.
      The Wall is automatically added to your StartUp folder.
    6. Double-click the Wall Program Referenced Object to start the Wall.

    • This installation does not alter your config.sys.
    • Wall upgrades can be installed over existing Wall installations.
    • No other steps (such as rebooting your computer) are required.

    Caution:
     This application requires the installation of "VROBJ.DLL".
    You can get a copy from...

    • Download VROBJ.DLL 2.1d (vrobj21d.zip)

Uninstallation
    How to uninstall the Wall...

    1. Close the Wall.
    2. Delete all files in the directory you have given to the Wall.
        ex. [x:\]del x:\Wall\*.*
    3. Remove the directory you have given to the Wall.
        ex. [x:\]rd Wall

    With the exception of EA data, all Wall files remain in the home subdirectory, including WALL.ini.

Functions

    Setting Options

      When you first start the Wall, the "simple" settings notebook appears. Check `Full Function' check box to see all settings options. The "Full Function" window will open automatically until unchecked.

      1. Select the bitmap directory:
        The Wall automatically uses the \os2\bitmap directory. You can select another directory by typing in full path of the directory or by using the `Drive' button and with `Directory' list. Either way you gets the same result.
      2. Select the bitmap subdirectory:
        You can also appoint the subdirectories by checking the `SubDirectory' check box, however, the selection of subdirectories will be automatically set under the parent directory which you select.
      3. Be sure to check the buttons located on the bottom of `Bitmap List'list box applied to the file types you may use for your desktop, in such as the use of `bmp', `gif', `jpg', `tif', `tga', `pcx' and `dib', otherwise such types of files might not be appeared on your desktop appropriately.
      4. Order your bitmaps:
        Your bitmaps will appear in alphabetical order when you check `Sequence'. Check `Random' if you prefer the random appearance of backgrounds.
      5. The `Preview Area' shows the currently selected bitmap.
        You can see the image in full by checking `Resize Bitmap' check box. If image file except .bmp is selected, then preview window shows only thumbnail.

    Types

      You can set the types of the Image File for each and every Image File you selected by selecting the `Image Type' spin button, such as Normal, Sized, Tiled. Please set the tile ratio when you select the `Sized'.

    Intervals

      You have 2 ways of set intervals.

      1. Select seconds, minutes or hours. Then use the slider to select the number of units(1-60) desired.
      2. You can set it by using the `Time selector'. It allows you to easy setting in each 10 minutes period up to one hundred twenty minutes (2 hours). Additionally, you can set it more than such two (2) hours by editing the Wall.ini (see `The Secret of Wall.ini'). There might be no theological limitation of the interval time if you challenge editing the Wall.ini.
      3. The set of the interval and the time remains will be shown in the each window: Interval; Remaining.

      Note:
       WALL.ini can be edited in a text editor. Additional time settings can be made this way.

    Editing parameters for the `TimeSelector'

      Clicking the `Modify' button opens the `Modify Timer Unit List' window. That lets you to edit the parameters of the `TimeSelector' by adding and/or eliminating the numbers being specified to allows the usable time in such TimeSelector function.
      The default number is;

        10m;20m;30m;40m;50m;1h;1h10m;1h20m;1h30m;1h40m;1h50m;2h.

      The word "m" means "minute". The word "h" stands "hour" (No seconds will be available in this setting.)
      Separate each parameter with a semicolon(;).
      For Example, you want to add 15minutes to these parameters,
      then;

        10m;15m;20m;30m;40m;50m;1h;1h10m;1h20m;1h30m;1h40m;1h50m;2h

    Zero Time Function

      1. Use `ZeroTimeFunction' to set up the Wall startup time on exact hour basis (xx:00).
        Let's say, you set the interval as 20min. and start the program at 09:23, the FIRST change will come at 09:40 rather than 09:43 and the program will change the bitmap every 20min. after then... such as 10:00, 10:20, 10:40 so on...

      2. If the set interval is over one hour, the FIRST change will be the next xx:00.
        Say, you set 1h20min. for the interval and start the program at 09:23, the FIRST change will be at 10:00 rather than 10:43. Then the program will change the bitmap every 1h20min.(80min.) after then.

    Settings of the Colors

      1. You can set your own color mix by moving each color sliders up and down. Watch the frame of the `Preview Area' until the desired color appears.
      2. Check the `Accept' button once you find the desired combination.
      3. Check the `Color Only' check box to change colors only. Background images will not appear using this option.
      4. Checking the `Random Color' check box allows the Wall to select colors at random.

    Startup Bitmap Function

      You can set startup image file.

      1. Check `Change at Startup' check box.
      2. Check `Startup Image' check box.
      3. There are 2 ways to set image.
        1. Type image file name with fullpath into entry field.
        2. Select image file at `Bitmap List' and push `Set' button.

    Set Active Individual Images

      You can maintain the individual settings in background color and image type to each bitmap you selected.

      1. Check the `Use Bitmap EA' check box.
      2. Select image file in the `Bitmap List Box'.
      3. Set Active Individual Image parameter.
        There are 3 ways to select Active Individual Image parameters.

        1. Choose tiled/normal/sized (select the Background Color as well when choosing Normal) click `Accept'. Now the individual settings you selected are maintained in the EA of the selected Image File.
        2. You can accomplish most of the above functions by using `Key Commands'. `Key Commands' are available when the focus is in the `Bitmap List' list box.
          Below are the 5 key commands you can use.

            1. Ctrl + C : Clear EA Data
            2. Ctrl + N : Set Normal mode
            3. Ctrl + S : Set Sized mode
            4. Ctrl + T : Set Tiled mode
            5. Ctrl + U : Change Use/Unused.

        3. Click mouse right button on `Preview Area' or file selected at `Bitmap List' list box, you can select parameter. You may set and register the setting of more than one bitmap files at one time by selecting them from the `Bitmap List'.
          You can only select following parameters with this way.

            1. Use Mode (Use/Unuse)
            2. Type (Normal/Tiled/Sized)
            3. Clear Data

    Muliple External Commands

      The Wall allows you to execute the option programs such as: *.exe, *.com, *.cmd, *.bat .
      You may add and delete those option programs by clicking `Detail' button besides the `Multiple External Commands' check box.

      This function has following parameters.

      1. `When Changed'
        The external commnad will run when the Wall changes the background image.
      2. `Fixed Time'
        The external command will run at exact time you set.
      3. `Enabled'
        You can set `Enabled/Disabled' by each external command.

    Sound

      You can play a wave file(`*.wav') files when the wall paper changes.

      1. Check the `Speaker Icon' to set "ON"; Uncheck to turn off sound. (The sound icon is blue when on, gray when off.)
      2. To use optional wave files(`*.wav'), select your own sound directory by typing the directory path into the "Sound Dir" field. When multiple .wav files reside in your subdirectory, the files are played in random.
      3. `Current' is the default setting.

    Exit After 1st Change Function

      You can exit the Wall after its first change.
      You can enable/disable this function with Exit1Chg.cmd.

      Usage:

      • Enable `Exit After 1st Change' Function
          [C:\WALL]Exit1Chg /E

      • Disable `Exit After 1st Change' Function
          [C:\WALL]Exit1Chg /D

      Note:
      You should exit the Wall before run Exit1Chg.cmd.

    External Image Viewer

      You can open selected image by external image viewer.

      • Set Image Viewer
        1. Click `Set' button for selecting image viewer.
        2. Select image viewer file.

      • Open Image
        1. Click mouse right button on `Preview Area' or file selected at `Bitmap List' list box.
        2. Select `Open' parameter.

      Note:
      Default viewer is OS/2 IB.EXE.

    The Secret of Wall.ini

      Wall.ini is the simple text file which you can edit by any editor you like.

      Caution:
      Be sure to back up WALL.ini before editing.
      Please do not touch it unless you must and know what you're doing.

      • UseEAArea: Set the field to use the EA area of bitmap file. No tests have done other than the `.COMMENT' field. So alteration is not recommended in this section.

    Wall Emergency Option Program (WallEmr)

      You can start the Wall Emergency Option Program ("WallEmr") at the same time the Wall starts by setting "ON" of the WallEmr button which is located the Speaker Icon. (see the use of WallEmr)

    Wall Emergency Option Program

      You can use it as you use the Wall....

    Termination

      Not recommended! But if you must....
      You can check out anytime you like by pressing "Exit".

      1. You can terminate the Wall by pressing the `Go ahead...' when you see the confirmation popup menu after you pressed the `Exit' button.
      2. You can cancel the such termination by pressing the `Stop...' when you see the confirmation popup menu. (Thank you)

    Other Functions

      1. Start the Wall minimized by checking the `Minimized Startup' check box.
      2. Change backgrounds at startup by checking `Change at Startup' check box. You can set startup image file (ref. Startup Bitmap Function).
      3. Use Windows-type bitmaps (rather than the OS/2) by checking `Use Windows Bitmap' check box. (Try this check box if you have a problem of viewing selected bitmaps.)
      4. Check the `Check Screen Size Everytime' check box when you change your desktop size (size of screen). Do not need this option to be checked otherwise.
      5. Check the `File Synchronize' check box if you want to synchronize indicate file name of `Bitmap List' when changed background.
      6. Check the `Max Size' check box if you wan to set maximum image file size. You can set size every 1KB (1024byte).
